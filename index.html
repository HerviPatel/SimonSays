Game rules

  • Click anywhere on the screen to start the game.
  • Remember the flashing button. Click the buttons in the same order.
  • Every upcoming level adds a new button. (i.g. Level 2 contains previous level 1 button + the new flashed button.)
  • If you miss any button from the sequence the game will end right away.
  • Every successful sequence will take you to another level. Best of luck!
